# 模块变更日志

每个功能模块一个文件，记录所有修改。格式：

```
### [YYYY-MM-DD] 操作摘要
- **文件**: 修改的文件路径
- **操作**: 具体做了什么
- **原因**: 为什么要改
- **影响**: 哪些功能/模块受影响
- **验证**: 如何确认改动正确
```
